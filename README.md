# Calculatrice-MVC (GUI)
Projet final fall semester 2025-2026

# Description
Ce projet est une implémentation d'une calculatrice simple en utilisant le modèle de conception MVC (Model-View-Controller). 

# Fonctionnalités
- Addition, soustraction, multiplication, division
- Racine carrée
- Exposant
- Mode
- Historique des calculs

# Technologies utilisées
- python
- customtkinter pour l'interface graphique
- bibliothèque math pour les opérations mathématiques
- eval pour l'évaluation des expressions

# Installation
1. Clonez le dépôt :
```bash
git clone https://github.com/djevenson/calculatrice_mvc
```
2. Accédez au répertoire du projet :
```bash
cd calculatrice_mvc
```
3. Installez les dépendances :
```bash
pip install -r requirements.txt
```
4. Exécutez l'application :
```bash
python main_interactive.py
```
