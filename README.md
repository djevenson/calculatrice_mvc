# Installation
Pour installer ce projet, suivez les étapes ci-dessous :
1. Clonez le dépôt sur votre machine locale :
   ```bash
   git clone
    ```2. Accédez au répertoire du projet :
   ```bash
    cd calculatrice_mvc
    ```3. Ouvrez le projet dans votre environnement de développement préféré (par exemple, Visual Studio Code, IntelliJ IDEA, etc.).
2. Assurez-vous d'avoir python installé et custumtkinter sur votre machine. Vous pouvez vérifier cela en exécutant les commandes suivants dans votre terminal :
    ```bash
     python --version
     pip show customtkinter
     ```5. Si vous n'avez pas custumtkinter installé, vous pouvez l'installer en utilisant pip :
    ```bash
     pip install customtkinter
     ```6. Une fois que vous avez installé les dépendances nécessaires, vous pouvez exécuter le projet en lançant le fichier principal (main_interactive.py) dans votre environnement de développement.


    



# Utilisation
Une fois que vous avez installé le projet, vous pouvez l'utiliser pour effectuer des calculs simples. Voici comment utiliser la calculatrice :
1. Lancez l'application.
2. Entrez les chiffres et les opérations(addition, soustraction, multiplication, division,exposant,racine carrée) que vous souhaitez effectuer.
3. Appuyez sur le bouton égal (=) pour obtenir le résultat.
4.vous pouver egalement utiliser les touches de votre clavier pour entrer les chiffres et les opérations.
5. ouvert et fermer les parentheses pour effectuer des calculs plus complexes.
6. r-->> radical , ** --> exposant , backspace pour effacer le dernier caractère entré , * --> multiplication , / --> division , + --> addition , - --> soustraction, . --> point pour les nombres décimaux, enfin les chiffres de 0 à 9 pour entrer les nombres.

# objective
L'objectif de ce projet est de créer une calculatrice simple en utilisant le modèle MVC (Model-View-Controller). Le modèle MVC est une architecture logicielle qui sépare les différentes responsabilités d'une application en trois composants principaux : le modèle, la vue et le contrôleur.
- Le modèle (Model) est responsable de la gestion des données et de la logique métier de l'application. Il contient les classes et les méthodes qui manipulent les données et effectuent les calculs nécessaires pour la calculatrice.
- La vue (View) est responsable de l'affichage de l'interface utilisateur. Elle contient les éléments graphiques tels que les boutons, les champs de saisie et les étiquettes qui permettent à l'utilisateur d'interagir avec la calculatrice.
- Le contrôleur (Controller) est responsable de la gestion des interactions entre le modèle et la vue. Il reçoit les entrées de l'utilisateur, les traite en utilisant le modèle et met à jour la vue en conséquence.
En utilisant le modèle MVC, nous pouvons créer une calculatrice modulaire et facilement maintenable, où chaque composant a une responsabilité claire et distincte. Cela facilite également la collaboration entre les membres du groupe, car chacun peut se concentrer sur une partie spécifique du projet.
